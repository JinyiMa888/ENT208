

# 简历优化网站 - ResumeBoost

## 整体设计
专业商务风格，以深蓝(#1E3A5F)和白色为主色调，类似LinkedIn的简洁大气风格。

## 页面结构

### 1. 首页 (Landing Page)
- **Hero区域**：标题"AI智能简历优化"，副标题说明核心功能，CTA按钮"开始优化简历"
- **优秀简历模板展示**：展示6-8个精选简历模板（技术类、市场类、金融类等），卡片式布局，可预览
- **500强公司招聘要求**：展示如Google、Apple、Microsoft、Amazon等公司的热门岗位招聘要求，包含岗位描述、关键技能、学历要求等
- **功能介绍**：三步流程图（上传简历 → AI分析匹配度 → 获取优化建议）

### 2. 用户认证
- 使用Lovable Cloud实现邮箱+密码注册登录
- 创建用户profiles表存储基本信息
- 登录后可保存历史分析记录

### 3. 简历优化工作台（核心功能，需登录）
- **左侧**：上传简历（支持PDF/DOCX）+ 输入期望职位和公司
- **右侧**：AI分析结果展示
  - **匹配度评分**：环形图展示总体匹配度（如78%）
  - **维度分析**：技能匹配、经验匹配、教育背景、关键词覆盖等各维度评分
  - **优化建议**：逐条列出需要修改的地方，包括添加关键词、调整表述、补充经验等
  - **优化后简历预览**：展示AI修改后的简历内容，高亮标注修改部分

### 4. 历史记录页面
- 查看过去的分析记录，包含时间、职位、匹配度评分

## 技术实现
- **前端**：React + Tailwind + shadcn/ui组件
- **后端**：Lovable Cloud（数据库 + Edge Functions + 认证）
- **AI分析**：通过Edge Function调用Lovable AI Gateway，分析简历与职位的匹配度并生成优化建议
- **文件上传**：Supabase Storage存储用户上传的简历文件

## 数据库表
- `profiles` - 用户基本信息
- `resume_analyses` - 分析历史记录（用户ID、职位、匹配度、建议、时间）
